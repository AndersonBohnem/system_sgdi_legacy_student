Como usar

Inclua o header em toda requisição.

Adicione o header X-API-Key em cada chamada: 

GET /api/v1/demandas
Host: localhost:5000
X-API-Key: sua-chave-aqui

Consulte todos os endpoints disponíveis na documentação Swagger.
